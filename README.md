## What is this tool?
This application helps users manage game files stored on external drives formatted with the exFAT file system. The PlayStation 5 uses this system for external storage. This tool reads the contents of your drive and generates a list of your installed games. It helps you keep track of your library without needing to connect the drive to the console.

## ⚙️ System requirements
To use this software, your computer needs these items:
* Microsoft Windows 10 or Windows 11.
* A USB port to connect your external drive.
* An external hard drive or solid-state drive formatted for use with a PS5.
* At least 50 megabytes of free space on your computer storage.

## 🔍 How to use the app
Follow these steps to scan your drive contents:
1. Select your drive letter from the dropdown menu in the app.
2. Click the "Scan Drive" button.
3. Wait for the progress bar to finish.
4. The app displays a table showing the title of each game, its file size, and the date it was copied to the drive.
5. You can export this list as a text file for your personal records by clicking "Save List."

## 🛠 Troubleshooting common issues
If the application fails to detect your drive, check these items:
* Ensure the drive is plugged into a USB 3.0 or higher port.
* Verify the drive is formatted as exFAT. Windows cannot see drives formatted in the proprietary Sony format if they contain encrypted system data.
* Restart the application after plugging in your drive.
* Ensure you have permission to read the drive contents. Some security software may block access to external USB storage devices. Add this app to your list of allowed programs if you face issues.

## 🖇 Frequently asked questions
Do I need to copy my games to my PC?
No, the tool reads the drive directly. No need to move files. 

Does this tool work with internal console storage?
No, this tool works only with external drives that connect via USB.

Can I use this to play games on my PC?
No, this tool only organizes and lists the files. It does not run or emulate PS5 games.

Is this safe for my drive? 
The tool uses a read-only mode to scan your files. It does not modify, delete, or change any data on your external drive. Your game library stays safe during the scan process.

Do I need to install drivers?
No, the application works immediately once you run the executable file. It uses standard Windows features to communicate with your hardware.

What happens if I unplug the drive during a scan?
The app will stop the scan and give an error message. Simply reconnect the drive and click "Scan Drive" again to restart the process.

Can I scan multiple drives at once?
The current version supports one drive at a time. To scan a different drive, finish the current task, unplug the first drive, plug in the new one, and refresh the dropdown menu.
